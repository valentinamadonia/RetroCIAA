AMP Chess TFT Screen (ILI9341 SPI)
=============================


Versión del ajedrez core vs. core para pantalla TFT ILI9341 (u$s20).
Necesita la librería también disponible, portada desde la versión para STM32 de [Tilen Majerle.](https://stm32f4-discovery.net/tag/tilen-majerle/)
Los cores de la EDU-CIAA se enfrentan al ajedrez usando la versión sin hash table, de [Andre Adrian](http://www.andreadrian.de/schach), del motor de ajedrez [micromax](http://home.hccnet.nl/h.g.muller/max-src2.html) y firmware bare-metal del [proyecto CIAA](https://github.com/ciaa/Firmware).

Otras versiones de EDUCIAA Chess disponibles [aquí](https://github.com/telmomoya/educiaa-bm/)
